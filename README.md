# Task Management Tool

A simple full-stack Task Management application built with Angular, Python, and SQLite.

##  Features
- Create, edit, delete tasks (CRUD)
- Real-time search filtering
- Task status toggle (open/completed)
- Custom "time ago" feature (no external libraries)

##  Tech Stack
- Frontend: Angular
- Backend: Python (http.server)
- Database: SQLite

##  Project Structure
- frontend/ → Angular UI
- backend/ → Python API + SQLite
